    --INFORMATION--
  scrapes data from indeed to generate a csv file that can be imported into a SQL table or PowerBI for analytic purposes on paying fairly. Uses 2 input files, one with a list of jobs and one with a list of cities or states.

    --INSTALL AND USAGE--
  first install the newest python version if not installed,
go thru setup and use normal settings EXCEPT add python to path AND disable path length limit.
once installation is complete use the installandrun.bat file the first time, then use the run.bat each time thereafter to generate a .csv file with the salary.
you can go into and edit the cities.txt and jobs.txt to vary where you look and for what jobs, if that city and job combination it will get the salary information for the whole state.
